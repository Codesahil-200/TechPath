# TechPath - IT Careers Learning Platform

TechPath is a modern web application designed to help users discover and learn about various IT roles. It features a career quiz, filtered roles exploration, and detailed roadmaps for each career path.

## Project Structure

This project has been simplified to 4 main files for ease of understanding and maintenance:

1. **index.html**: The main landing page and career discovery interface.
2. **admin.html**: The administrator dashboard for viewing user leads and data export.
3. **app.js**: The core logic for filtering, quiz functionality, and data persistence using IndexedDB.
4. **tailwind.css**: The styles for the entire application, including utility-first Tailwind CSS and custom beautifications.

## Features

- **Career Discovery**: Explore various IT roles with detailed descriptions and salary ranges.
- **Career Quiz**: Take a short quiz to find which IT career path suits you best.
- **Detailed Roadmaps**: Every role comes with a step-by-step learning guide and resources.
- **User Registration**: Users can register their interest to view detailed career paths.
- **Admin Dashboard**: Securely view and export user leads locally in your browser.

## Technologies Used

- **HTML5 & Vanilla JavaScript**: Core structure and logic.
- **Tailwind CSS**: Modern UI styling.
- **IndexedDB**: Local browser database for persistent data storage without requiring a backend server.
- **Lucide Icons**: Beautiful iconography.

## Getting Started

Simply open `index.html` in any modern web browser to start exploring!
