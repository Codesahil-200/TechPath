# 🚀 TechPath - IT Careers Learning Platform

TechPath is a premium, modern web application designed to help enthusiasts discover and navigate the landscape of IT careers. It features a career-matching quiz, interactive role exploration, and detailed learning roadmaps.

---

## ✨ Key Features

- **🎯 Career Discovery**: Explore curated IT roles with detailed descriptions and salary insights.
- **🧠 Interactive Quiz**: Take a smart quiz to identify which tech path best fits your personality and logic.
- **🗺️ Learning Roadmaps**: Step-by-step guides from Junior to Lead roles with curated resource lists.
- **📱 Responsive Design**: Fully responsive UI crafted with Tailwind CSS for a seamless desktop and mobile experience.
- **🔐 Admin Dashboard**: Secure browser-based portal to view and export user leads without needing a backend.

## 🛠️ Technical Stack

- **HTML5 & Vanilla JS**: Core application structure and logic.
- **Tailwind CSS**: Modern utility-first styling with premium glassmorphism effects.
- **IndexedDB**: Persistent local browser storage for high-speed data handling without a server.
- **Lucide Icons**: Clean and modern iconography.

---

## 📂 Project Structure

This project is built for simplicity and maximum performance:
1. `index.html`: The main customer-facing career discovery interface.
2. `admin.html`: The administrator dashboard for data management (Password: `admin123`).
3. `app.js`: Core engine for quiz logic, filtering, and data persistence.
4. `tailwind.css`: Production-ready design system and utility styles.
